# オンゲキ 設置店舗マップ

[オンゲキ](https://ongeki.sega.jp/) の全国設置店舗を地図上で探せる Web マップ。

ピンをタップすると店舗名・住所と Google マップへのリンクが表示されます。

データは [SEGA AM Location](https://location.am-all.net/alm/location?gm=88) の公開情報をもとに、週次で自動更新されます。
